# SQL – Grundlagen

## Was ist SQL?
SQL wird verwendet, um Daten in relationalen Datenbanken zu lesen und zu verändern.

## Wichtige Begriffe
- Datenbank = Sammlung von Tabellen
- Tabelle = strukturierte Datensammlung
- Zeile = ein Datensatz
- Spalte = eine Eigenschaft
- Primärschlüssel = eindeutige ID

## CRUD
- Create = `INSERT`
- Read = `SELECT`
- Update = `UPDATE`
- Delete = `DELETE`
