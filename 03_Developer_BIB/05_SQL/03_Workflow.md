# SQL – Workflow

1. Ziel festlegen: lesen, einfügen, ändern oder löschen
2. Richtige Tabelle auswählen
3. Benötigte Spalten bestimmen
4. Bei Änderungen eine `WHERE`-Bedingung festlegen
5. Abfrage vorbereiten
6. Abfrage ausführen
7. Ergebnis prüfen

## Merksätze
- `INSERT INTO` = Daten hineinlegen
- `SELECT ... FROM` = Daten herausholen
- `UPDATE` = Daten ändern
- `DELETE FROM` = Daten entfernen
