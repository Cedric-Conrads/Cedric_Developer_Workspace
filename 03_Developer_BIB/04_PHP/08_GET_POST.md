🟦 GET

GET = holen gehen.

Ich hole Informationen.

Beispiel:

bearbeiten.php?artikelnummer=1002

PHP:

$_GET["artikelnummer"]
🟩 POST

POST = Daten an PHP schicken.

Der Benutzer hat etwas geändert.

PHP weiß das noch nicht.

Deshalb schicken wir die neuen Werte mit:

$_POST["name"]
$_POST["bestand"]
$_POST["lagerort"]
