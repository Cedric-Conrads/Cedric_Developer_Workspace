# PHP – Grundlagen

## Was ist PHP?
PHP läuft auf dem Server und verarbeitet Anfragen.

## Grundidee
```text
Browser
↓
PHP
↓
Datenbank
↓
PHP
↓
HTML
↓
Browser
```

## Wichtige Begriffe
- `<?php ... ?>` = PHP-Bereich
- Variable beginnt mit `$`
- `$_POST` = Formulardaten aus einer POST-Anfrage
- `$_GET` = Daten aus der URL
- `echo` = Text oder HTML ausgeben
- `header()` = Weiterleitung
- `exit` = Skript stoppen
- `session_start()` = Session öffnen oder starten

## Merksatz
Frontend fragt. Backend verarbeitet und entscheidet.
