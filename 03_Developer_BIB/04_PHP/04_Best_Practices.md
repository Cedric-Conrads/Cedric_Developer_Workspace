# PHP – Best Practices

- Eingaben prüfen und bereinigen.
- Prepared Statements verwenden.
- Passwörter nie im Klartext speichern.
- `password_hash()` und `password_verify()` nutzen.
- Nach `header()` immer `exit` verwenden.
- Datenbankverbindung später in eine eigene Datei auslagern.
- Fehlermeldungen lesen und schrittweise debuggen.
- Browser und Datenbank niemals direkt miteinander verbinden.
