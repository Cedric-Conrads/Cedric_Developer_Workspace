# git commit

Syntax:
`git commit -m "Nachricht"`

`-m` steht für message.

Beispiel:
`git commit -m "Initialen Developer Workspace erstellt"`



Mein erster Git-Commit

git commit -m "Initialen Developer Workspace erstellt"

↓

Erstellt den ersten dauerhaft gespeicherten Meilenstein
des Projekts.

Ein Commit speichert:

- den aktuellen Stand
- den Autor
- Datum und Uhrzeit
- die Commit-Nachricht

Merksatz:

📸 Ein Commit ist ein dauerhaft gespeichertes Foto
des aktuellen Projekts.
