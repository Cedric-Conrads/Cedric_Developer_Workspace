# Git – Wichtige Befehle

## git init
Macht einen Ordner zu einem Git-Repository.

## git status
Zeigt den aktuellen Zustand.

## git add
Bereitet Änderungen für den nächsten Commit vor.

git add .

↓

Git verschiebt die Änderungen NICHT ins Repository.

Git verschiebt sie in die Staging Area.

git status zeigt danach:

Changes to be committed

↓

Diese Änderungen werden beim nächsten Commit gespeichert.
