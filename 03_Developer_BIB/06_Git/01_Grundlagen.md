# Git – Grundlagen

## Was ist Git?
Git ist ein Versionsverwaltungssystem.

## Warum gibt es Git?
- Änderungen nachvollziehen
- Zu älteren Versionen zurückkehren
- Im Team arbeiten

## Merksätze
- git init erstellt ein Git-Repository.
- Git speichert nicht automatisch.
- Du entscheidest, wann ein Commit erstellt wird.
