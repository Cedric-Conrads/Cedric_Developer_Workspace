# JavaScript – Best Practices

- Aussagekräftige Variablennamen verwenden.
- Eingaben immer prüfen.
- Funktionen klein und verständlich halten.
- Wiederholten Code in Funktionen auslagern.
- `const` verwenden, wenn keine Neuzuweisung nötig ist.
- Fehler mit `console.log()` und Entwicklertools prüfen.
- Nicht blind Code übernehmen, sondern jede Zeile verstehen.
