# JavaScript – Wichtige Befehle

- `document.getElementById()` = Element über ID finden
- `.value` = Inhalt eines Eingabefeldes holen
- `.trim()` = Leerzeichen am Anfang und Ende entfernen
- `.textContent` = sichtbaren Text ändern
- `addEventListener()` = auf eine Aktion warten
- `if / else` = Bedingungen prüfen
- `return` = Funktion an dieser Stelle stoppen
- `createElement()` = neues HTML-Element erstellen
- `appendChild()` = Element einfügen
- `remove()` = Element entfernen
- `querySelector()` = Element mit CSS-Schreibweise finden
- `push()` = Eintrag zu einem Array hinzufügen
- `forEach()` = jeden Eintrag durchgehen
- `localStorage.setItem()` = im Browser speichern
- `localStorage.getItem()` = aus Browser-Speicher laden
- `JSON.stringify()` = Daten einpacken
- `JSON.parse()` = Daten auspacken
