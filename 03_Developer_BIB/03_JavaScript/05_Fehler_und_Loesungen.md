# JavaScript – Fehler und Lösungen

## Element ist `null`
Die ID im HTML stimmt nicht mit JavaScript überein oder das Script wird zu früh geladen.

## Button reagiert nicht
- `addEventListener` prüfen
- ID prüfen
- Script-Verbindung prüfen

## Zahlen werden als Text verbunden
`Number(...)` verwenden.

## Daten verschwinden nach Neuladen
Sie liegen nur in Variablen. Für Browser-Speicherung `localStorage` verwenden.
