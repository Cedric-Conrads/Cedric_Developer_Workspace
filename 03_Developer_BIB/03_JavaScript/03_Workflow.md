# JavaScript – Workflow

1. HTML-Element finden
2. Element in einer Variable speichern
3. Auf ein Ereignis warten
4. Eingaben auslesen
5. Eingaben prüfen
6. Daten verarbeiten
7. Ausgabe im Browser ändern
8. Bei Bedarf Daten speichern

## Beispiel
```text
Button-Klick
↓
Eingaben holen
↓
if prüft
↓
Daten verarbeiten
↓
Anzeige aktualisieren
```
