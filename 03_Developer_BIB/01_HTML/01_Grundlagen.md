# HTML – Grundlagen

## Was ist HTML?
HTML steht für **HyperText Markup Language** und baut den Inhalt sowie die Struktur einer Webseite.

## Grundidee
- HTML = Inhalt und Struktur
- CSS = Aussehen und Design
- JavaScript = Verhalten und Funktionen

## Grundgerüst
```html
<!DOCTYPE html>
<html lang="de">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Seitentitel</title>
</head>
<body>

</body>
</html>
```

## Merksätze
- `<head>` enthält Informationen über die Seite.
- `<body>` enthält alles, was im Browser sichtbar ist.
- HTML beschreibt, **was** auf einer Seite vorhanden ist.
