# HTML – Workflow

1. Ziel der Seite festlegen
2. Inhalte und Bereiche planen
3. HTML-Grundgerüst erstellen
4. Semantische Elemente verwenden
5. IDs und Klassen sinnvoll vergeben
6. CSS und JavaScript verbinden
7. Im Browser testen
8. Fehler Schritt für Schritt prüfen

## Merksatz
Erst Struktur bauen, danach gestalten und programmieren.
