# HTML – Fehler und Lösungen

## JavaScript funktioniert nicht
**Prüfen:**
- Ist `script.js` richtig verbunden?
- Stimmt `src` und nicht versehentlich `scr`?
- Liegt die Datei im richtigen Ordner?

## CSS wird nicht geladen
**Prüfen:**
- Stimmt der Pfad in `href`?
- Heißt die Datei wirklich `style.css`?

## JavaScript findet ein Element nicht
**Ursache:** ID in HTML und JavaScript stimmen nicht exakt überein.

## PHP erhält keine Formulardaten
**Prüfen:**
- Hat das Feld ein `name`-Attribut?
- Stimmen `action` und `method`?
