# HTML – Wichtige Elemente

## Texte
- `<h1>` = wichtigste Überschrift
- `<h2>` bis `<h6>` = weitere Überschriften
- `<p>` = Textabsatz
- `<span>` = kleiner Bereich innerhalb eines Textes

## Struktur
- `<main>` = Hauptinhalt
- `<section>` = thematischer Abschnitt
- `<div>` = neutraler Container
- `<header>` = Kopfbereich
- `<footer>` = Fußbereich

## Eingaben
- `<form>` = Formular
- `<input>` = Eingabefeld
- `<textarea>` = größeres Textfeld
- `<button>` = Button
- `<label>` = Beschriftung eines Feldes

## Verbindungen
- `<a href="">` = Link
- `<img src="">` = Bild
- `<link rel="stylesheet" href="style.css">` = CSS verbinden
- `<script src="script.js"></script>` = JavaScript verbinden

## Attribute
- `id` = eindeutiger Name, oft für JavaScript
- `class` = wiederverwendbarer Name, oft für CSS
- `name` = Schlüssel beim Absenden eines Formulars
- `placeholder` = Beispieltext im Eingabefeld
