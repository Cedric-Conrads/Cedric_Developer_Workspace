# HTML – Best Practices

- Nur eine Hauptüberschrift `<h1>` pro Seite verwenden.
- Aussagekräftige Namen für `id` und `class` wählen.
- Eingabefelder mit `<label>` beschriften.
- Elemente sauber einrücken.
- Semantische Elemente wie `<main>` und `<section>` verwenden.
- Bilder mit einem passenden `alt`-Text versehen.
- JavaScript meist vor `</body>` einbinden.
- Dateinamen klein und ohne Leerzeichen schreiben.
