# CSS – Best Practices

- Klassen statt vieler IDs für Design verwenden.
- Wiederholungen vermeiden.
- Mobile Ansicht mitdenken.
- Einheitliche Abstände benutzen.
- Sinnvolle Klassennamen wählen.
- `box-sizing: border-box` global setzen.
- Große CSS-Dateien später in Bereiche aufteilen.
- Keine unnötigen Inline-Styles verwenden.
