# CSS – Wichtige Eigenschaften

## Abstände
- `margin` = Außenabstand
- `padding` = Innenabstand

## Größe
- `width` = Breite
- `height` = Höhe
- `max-width` = maximale Breite

## Farben und Design
- `color` = Textfarbe
- `background` = Hintergrund
- `border` = Rahmen
- `border-radius` = runde Ecken
- `box-shadow` = Schatten

## Layout
- `display: flex` = flexible Ausrichtung
- `display: grid` = Raster
- `justify-content` = horizontale Verteilung
- `align-items` = Ausrichtung auf der anderen Achse
- `gap` = Abstand zwischen Elementen

## Interaktion
- `:hover` = Stil beim Darüberfahren
- `transition` = weicher Übergang
- `cursor: pointer` = Mauszeiger als Hand

## Responsive Design
```css
@media (max-width: 700px) {
    /* Regeln für kleine Bildschirme */
}
```
