# CSS – Fehler und Lösungen

## Stil wird nicht angewendet
- Ist die CSS-Datei verbunden?
- Stimmt der Selektor?
- Stimmt Groß- und Kleinschreibung?
- Überschreibt eine spätere Regel den Stil?

## Layout bricht auseinander
- Breiten, `padding` und `box-sizing` prüfen.
- Flexbox- oder Grid-Einstellungen prüfen.

## Mobile Ansicht ist schlecht
- Media Query ergänzen.
- Feste Breiten durch `max-width` oder Prozentwerte ersetzen.
