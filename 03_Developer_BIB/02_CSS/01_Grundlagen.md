# CSS – Grundlagen

## Was ist CSS?
CSS steht für **Cascading Style Sheets** und bestimmt das Aussehen einer Webseite.

## Beispiel
```css
body {
    background: #111827;
    color: white;
}
```

## Aufbau einer Regel
```css
selektor {
    eigenschaft: wert;
}
```

## Merksätze
- HTML baut das Element.
- CSS gestaltet das Element.
- Eine Klasse beginnt in CSS mit `.`.
- Eine ID beginnt in CSS mit `#`.
